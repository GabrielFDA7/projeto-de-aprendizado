Projeto 2 - Verificador de Disponibilidade

Projeto desenvolvido em VBA com integração com o SAP, utilizado na SOTREQ. O projeto foi feito com o intuito de automatizar o processo de verificação de disponibilidade de peças vendidas pela SOTREQ.

É uma planilha do Excel, em que o usuário insere o código da peça e os scripts criados em VBA são executados para verificar a disponibilidade da peça no SAP.

A principal vantagem desse projeto é a economia de tempo, já que o processo manual levava em média 5 minutos por peça, enquanto o processo automatizado leva apenas alguns segundos.

Além disso, a transação do SAP usada nesse projeto (YLP_PARTS) nunca havia sido integrada com o Excel antes, o que torna esse projeto inovador dentro da empresa.
